# Дневник чтения

Личное PWA-приложение для учёта прочитанных книг, прогресса и статистики.
Тёмный премиум-интерфейс, работает офлайн, ставится иконкой на iPhone и Mac.

## Возможности
- Главная-витрина: «сейчас читаю», серия дней, кольцо цели, тепловая карта
- Библиотека с обложками и фильтрами по статусу
- Карточка книги: прогресс, оценка 1–10, темп и прогноз, цитаты
- Добавление книги с авто-поиском обложки (Open Library)
- Умная статистика: авто-инсайты, графики, жанры, топ-авторы, рекорды, цель года
- Календарь чтения
- Бэкап и восстановление данных в файл

## Технологии
Чистый JavaScript (ES-модули), без сборки и зависимостей. IndexedDB для хранения,
Service Worker для офлайна, ручные SVG-графики.

## Установка на устройство
Открыть сайт в Safari → «Поделиться» → «На экран Домой».

## Данные
Все данные хранятся локально на устройстве (IndexedDB). Делайте резервные копии
через «Настройки → Сохранить копию данных».
